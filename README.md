# React-Starter-Skeleton

⚛️ React skeleton with React-Router, Redux, Redux-Persist, Axios and more!

## Dependencies

* react
* react-dom
* react-router, react-router-dom, react-router-redux
* redux, redux-persist, redux-thunk
* axios

## Installation

```
npm i
```

```
npm start
```

### Or

```
yarn install
```

```
yarn start
```
